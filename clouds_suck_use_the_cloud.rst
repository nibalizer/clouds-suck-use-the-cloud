Why you should use a cloud
==========================


* On Demand Resources
* Autoscaling
* Decentralized Team
* Not having to deal with hardware
* Hardware refreshed by Provider
* Future has new tech in it (containers)
* Cloud security team
  * Xen Vuln
  * DMCA
  * Network security
  * Compliance
* Integrations
* DBaaS and other \*aaS
  * DNS
* Object Store
